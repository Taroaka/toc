# Requirements: YouTube サムネイル用プロンプト生成エージェントを追加する

## 背景 / 問題

- `output/<topic>_<timestamp>/` 直下には物語動画の成果物が作られるが、
  YouTube サムネイル用のプロンプトを安定して作る専用エージェントが未定義。
- 現状は物語名ごとに手動でサムネ文言や背景演出を考える必要があり、
  文字の可読性・クリック性・YouTube 向け画角の配慮がブレやすい。
- 物語動画のサムネは「作品名を大きく見せる装飾文字」と
  「物語らしさが一目で伝わる背景」を両立する必要がある。

## ゴール

- ユーザー指示で起動できる `YouTube Thumbnail Prompt Writer` エージェントを追加する。
- エージェントは、物語名を必須入力として受け取り、
  必要なら `output/<topic>_<timestamp>/` の成果物を参照して、
  YouTube サムネイル用の構造化プロンプトを生成してユーザーへ返す。
- 出力は「そのまま画像生成ツールへ貼れる最終プロンプト」を含み、
  16:9 / 高解像度 / 文字可読性 / 余白 / クリック性の条件を明示する。

## 非ゴール / 除外

- エージェント自身が画像生成 API を呼ぶこと
- サムネ画像を自動保存すること
- 既存 run dir へ必ず `thumbnail_prompt.md` を書き出すこと

## 成功指標（MVP）

- `.claude/agents/` に新エージェント定義が追加される
- `docs/implementation/agent-roles-and-prompts.md` に責務が追記される
- 出力フォーマットとして、最低限次を返せる
  - 物語名
  - 推奨サイズ（16:9, 1280x720 以上）
  - デザイン方針
  - 最終プロンプト
  - 追加制約 / negative guidance
- 例として「桃太郎」「浦島太郎」のような昔話で使える
  装飾文字中心のサムネプロンプトが成立する
