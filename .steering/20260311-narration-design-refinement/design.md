# Narration Design Refinement Design

## 変更方針

調査結果をふまえ、ナレーションを次のように再定義する。

- 目的は、画面説明ではなく **観客の理解順序を整えること**
- 基本は **映像が見せる / ナレーションが見えない層を補う**
- 強いナレーションは、少なくとも次のいずれかを追加する:
  - 時間
  - 内面
  - 因果
  - 視点
  - 世界のルール / 禁忌
  - 半歩だけ抽象化した意味

## ルール化

### Must

- 原則として各ナレーション行は、映像だけでは分かりにくい情報を最低1つ追加する
- 視覚で一目で分かる物理行動や感情は、まず映像にやらせる
- ナレーションが無くても scene の大意は通る状態を理想とする

### Should

- 映像と完全一致させるより、補完関係を優先する
- 終盤の学び/余韻では、出来事から半歩だけ抽象化して満足感を作ってよい
- 昔話/伝承系では、禁忌・運命・因果を言語で補いやすい

### Avoid

- 画面の物理行動をそのまま読み上げるだけの行
- 制作側の専門語や設計思想の露骨な説明
- 毎行で抽象化しすぎて、物語の進行が止まること
