# Requirements

## Context

- ユーザーの目的は、ToC の `output/` にある画像や動画をスマホからいつでも個人閲覧できるようにすること
- 共有や通知ではなく、自己利用の閲覧基盤が必要
- 既に `CommandMate` は遠隔操作用に整備済みで、今回は成果物閲覧レイヤーを補完したい

## Problem

- `output/` 配下の成果物はローカルに散在しており、スマホから快適に確認しづらい
- Obsidian / Notion はメディア閲覧の主用途に向かず、フル尺動画や大量画像の確認に弱い
- 毎回手動でファイルを拾う運用だと継続しない

## Goals

- Immich を ToC の個人用メディア閲覧基盤として導入するための実行手順を root に追加する
- `output/` 配下の画像・動画を Immich へ同期するスクリプトを用意する
- 最小限の運用で、手動同期と将来の定期同期の両方に対応できる形にする

## Non-Goals

- Immich 自体の Docker コンテナ起動をこのターンで完了させること
- Tailscale のセットアップ
- `output/` の構造変更

## Acceptance Criteria

1. root 直下に Immich 導入メモが追加される
2. `output/` のメディアを Immich に同期するスクリプトが追加される
3. セットアップ手順、同期対象、手動実行コマンド、将来の自動化方針が明文化される
