# Requirements

## Goal

Russell の感情円環 / core affect を、既存の 6 つの感情曲線を置き換えずに、`story` / `script` 設計で参照できる補助レイヤーとして追加する。

## Requirements

1. Hollywood が Russell を明示採用していると断定せず、実証がある範囲と推論の範囲を分けて記述する
2. 既存の `6 arcs` は作品全体のマクロ設計として残し、scene / cut 単位では valence / arousal を使う方針を追加する
3. 台本設計では「意図した感情 (`intended`)」を正本にし、視聴者が実際に感じる感情 (`experienced`) と混同しない
4. 映画産業における近接事例として、少なくとも次を参照できるようにする
   - 映画の emotional arc と商業成果の分析
   - トレーラーの arousal 設計
   - greenlight / audience prediction のデータ活用
5. `story` / `script` / playbook / template の最低限の契約に、scene-level の affect target を追加する
6. 新しい guidance は既存フローを壊さず、未使用でも通る optional layer として導入する
