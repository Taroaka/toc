# Research Method: Deep Primary First

## Use when
- 原典・一次資料を優先して、後段で解釈を重ねたいとき

## Steps
1. 原典/一次資料を収集して短要約を作る
2. 学術/百科/二次資料で解釈差分を集める
3. 原話・版差・根拠抜粋・未検証点を `story_materials` 中心に `research.md` へ統合する
4. 使い所の見立てがある場合だけ `suggested_story_usage` に任意メモとして残す

## Output contract
- `story_materials.canonical_story_dump`
- `story_materials.chronological_events`（20+）
- `source_inventory` / `source_passages`
- `variants` / `conflicts`
- `facts.items`
- `handoff_to_story`
