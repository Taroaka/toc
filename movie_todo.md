# Movie TODO: 視覚化価値パートの導入と浦島太郎・竜宮城探索シーケンス

## 目的

物語本筋の説明だけではなく、動画生成AIだからこそ実写風に視覚化する価値が高いパートを設計に取り込む。
特に、昔話や神話に含まれる非現実的な場所・物・現象を、物理セット不要の強みを活かして壮大に映像化し、
視聴者の「見てみたい」を引き出す中盤の価値ピークを作る。

## 今回の方針

- [x] Deep Research エージェントと台本作成エージェントの間に、`視覚化価値想像エージェント` を追加する
- [x] このエージェントは、物語の中の非現実的要素を抽出し、
  現代の動画生成AIで実写化したときに強い視覚価値になるパートを提案する
- [x] 提案する価値パートは、動画の最序盤・最終盤ではなく、
  **全体の 20% - 80% の間**に差し込む前提で設計する
- [x] 1つの価値パートは **4-6 カット**で構成し、
  **各カット 4 秒固定・ナレーションなし**で視覚的満足を作る

## 視覚化価値想像エージェントの思考ルール

### 役割

物語の中にある「読者は知っているが、細部は見たことがないもの」を抽出し、
動画生成AIで視覚化したときに、視聴維持と満足度を上げるパートとして定義する。

### 思考のコツ

- 物語の中の **非現実的な場所 / アイテム / 現象 / 禁忌** を抽出する
- それらを、最新の動画生成AIで **実写風・映画品質** で見せたときに、
  視聴者が「そこをもっと見たい」と思うかで判断する
- 映画の実制作なら高価なセットや大規模VFXが必要なものほど優先度が高い
- 単なる背景ではなく、**探索したくなる / 触れたくなる / 開けたくなる / 留まりたくなる**
  ものを優先する
- 画面内の説明文に頼らず、**形 / 光 / 動き / 機構 / ショー性** で価値を成立させる

### 出力要件

- [x] 価値パート名
- [x] その価値パートを入れる理由
- [x] 動画全体のどこに差し込むか（20% - 80% の範囲）
- [x] 4-6 カットの探索 / 体験シーケンス
- [x] 各カット:
  - 4 秒固定
  - ナレーションなし
  - 主役となる場所 / 物 / 現象が明確
  - 何を感じさせたいかが1文で分かる

## 台本エージェントへの指示

- [x] 台本エージェントは `視覚化価値想像エージェント` の提案を参照して script を作る
- [x] 価値パートは本筋を止める寄り道ではなく、
  **中盤で視聴者に報酬を与える映像シーケンス** として使う
- [x] 価値パートは narration-driven ではなく visual-driven で扱う
- [x] そのため、対象カットではナレーションを空にし、
  映像の連続だけで満足感を作る

## 浦島太郎への適用

### 採用する価値パート

- [x] 価値パート名: `竜宮城の中身をPOVで探索するシーケンス`
- [x] 配置: 動画中盤（全体の 40% - 65% 付近を想定）
- [x] 目的:
  - 浦島太郎という題材で最も「見たいのに詳細が語られていない」要素を報酬化する
  - 乙姫に会う前に、竜宮城そのものの魅力と異界性を体験させる
  - 視聴者に「昔話の説明」ではなく「異世界を探索する映像体験」を与える

## 竜宮城探索シーケンス案（6カット、各4秒、ナレーションなし）

### Cut 1: 珊瑚の大門

- 内容:
  海中の暗がりの先で、巨大な珊瑚と真珠光沢の骨組みでできた竜宮城の門が開く。
  POVでゆっくり接近し、門の隙間から内側の光が漏れてくる。
- 狙い:
  「ここはただの海底ではなく、別の法則が支配する宮殿だ」と一目で分からせる

### Cut 2: 泡の回廊

- 内容:
  門を抜けると、天井近くに泡の灯りが連なり、床や壁に水の反射が揺れる長い回廊を進む。
  珊瑚の柱の間を小魚の群れが横切る。
- 狙い:
  竜宮城が建築物でありながら、生きた海そのものでもあることを見せる

### Cut 3: 魚群の大広間

- 内容:
  回廊の先に一気に空間が開け、巨大な大広間へ出る。
  頭上には渦を巻く魚群、中央には潮流でかたちを変える水の柱、床には金と貝殻の装飾が光る。
- 狙い:
  「竜宮城の中身」を強い視覚報酬として提示し、中盤の価値ピークを作る

### Cut 4: 海の宴の間

- 内容:
  次の間では、無人なのに宴の準備が整っており、
  発光する器、ゆらめく海藻の飾り、見たことのない海中料理が長い卓上に並ぶ。
  POVで卓の横をゆっくり進む。
- 狙い:
  乙姫に会う前に、「ここは招かれた者を歓迎し、帰りたくなくさせる場所だ」と伝える

### Cut 5: 時を忘れる奥廊下

- 内容:
  華やかな広間の先に、急に静かな奥廊下が現れる。
  天井の水面光がゆっくり脈打ち、遠景には薄い霧のような泡が漂い、
  時間感覚が曖昧になる空気をPOVの前進で見せる。
- 狙い:
  竜宮城の魅力を「豪華さ」だけでなく、「時間がずれる異界性」として深める

### Cut 6: 乙姫の間の入口

- 内容:
  最奥部で、生きた珊瑚の扉が水圧の変化で静かに開き、
  内側から柔らかな金色の光が差し込む。
  扉の向こうに乙姫の間の気配だけを見せ、入る直前でカットを終える。
- 狙い:
  探索シーケンスをドラマへ自然に接続し、次の登場に期待を持たせる

## 実制作メモ

- [x] 各カット 4 秒固定
- [x] ナレーションなし
- [x] 音は環境音と軽い幻想的BGMだけで成立させる

  - `otohime_chamber_entrance`

## 期待される効果

- 昔話の既知性を、映像体験の新しさで上書きできる
- 中盤に「見たかったものを見せる」報酬を置ける
- 低予算の実写では不可能な規模感を、生成AI前提で価値化できる
- 乙姫登場前の期待を高め、本筋への接続も強くなる
