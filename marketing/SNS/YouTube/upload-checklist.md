## B. タイトル設計

> 詳細な設計ガイド・パワーワード辞典・海外AI動画の実例は `title-design.md` を参照。

### 基本公式

```
[物語名（先頭15文字以内）] ｜ [感情フック] [具体要素] 【AI映像化/AI民話】
```

**文字数ルール**: スマホで省略されない **28〜32文字以内** に主要部分を収める。

### 浦島太郎 候補タイトル（コピーして使う）

```
浦島太郎｜AIが語る古の伝説【没入型アニメーション】      ← 28文字
【衝撃】浦島太郎 - 玉手箱を開けた本当の理由【AI民話】  ← パワーワード型
浦島太郎｜AIが描く3000年の孤独【完全映像化】           ← 感情フック型
```

### タイトルのルール（要点）

- 先頭15文字以内に「浦島太郎」を入れる（アルゴリズムが左側を優先）
- パワーワードは1〜2個（多すぎると信頼性が下がる）
- 感情ワード例: 「衝撃」「切ない」「永遠の代償」「真実」「孤独」
- 損失回避フレーミングが強い: 「失ったもの」「代償」「消えた〇〇」
- 数字を入れると具体性が増す: 「3000年の孤独」「300年後の真実」
- 釣りタイトル厳禁（視聴維持率が落ちYouTubeに嫌われる）

## F. アップロード設定

- [ ] 公開設定: 単発公開戦略に従い、初回は `火曜 19:00 JST` を基準に設定
- [ ] 「子ども向けコンテンツ」: いいえ（制限が少ない）
- [ ] 再生リスト: 「日本民話シリーズ」に追加（なければ作成）
- [ ] 字幕: 日本語字幕を自動生成のまま or 手動で修正
- [ ] エンドカード: 最後15秒に次の動画 or チャンネル登録ボタンを設定
- [ ] カード: 動画中盤に「シリーズ再生リスト」カードを挿入

補足:

- browser-use は設定漏れの点検には使ってよい
- 実際の公開確定は人間確認を挟む
- 初回公開の時刻設計は `publishing-strategy-urashima.md` を優先する

## G. 公開後24時間アクション（重要）

YouTubeはアップロード後24〜48時間の反応率を見て長期推薦を決める。

- [ ] 公開と同時にX（Twitter）でシェア（サムネ画像付き）
- [ ] Instagramにサムネ or ショートクリップを投稿
- [ ] コメント欄に「見どころ」や「制作の裏話」を自分で先に書く（コメントの呼び水）
- [ ] 24時間後にYouTube Studioで以下を確認:
  - CTR（目標4%以上）
  - 視聴維持率（目標45%以上）
  - インプレッション数
- [ ] 説明欄のリンクがLPに正しく飛ぶか確認
