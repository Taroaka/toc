最終更新: 2026-04-11  
用途: 浦島太郎以降の全動画に共通して使う。タイトルを考えるときは必ずここを参照する。

## 1. タイトルの基本ルール（研究ベース）

### 文字数

| 表示環境 | 推奨文字数 |
|---------|-----------|
| スマホ（最も重要） | **28文字以内**（これ以上は省略される） |
| PC | 〜60文字まで表示可能 |
| 検索結果 | 前半50文字が優先表示される |

**結論**: 主要キーワード + 感情フックを**28〜32文字以内**に収め、補足情報は後半に置く。

### キーワード配置の鉄則

```
先頭15文字 ← ここに最重要キーワードを入れる（検索・サイドバーで最初に見える）
```

- YouTube アルゴリズムはタイトルの左側を優先的にスキャンする
- 視聴者も左から読む→最初の単語が「クリックするかどうか」を決める

### 損失回避フレーミング

「メリットを伝える」よりも「失うことへの恐れを煽る」方がCTRが上がる。

| 弱い（ベネフィット訴求） | 強い（損失回避） |
|------------------------|----------------|
| 浦島太郎の魅力を解説 | 浦島太郎が失ったもの |
| 竜宮城の美しさ | 竜宮城に消えた300年 |
| AI動画の作り方 | AIを使わないと損する理由 |

## 2. タイトル構成パターン（テンプレート集）

### パターン A: 固有名詞 + 感情フック + 補足

```
[物語名]｜[感情ワード][具体的な何か]【補足情報】
```

例:
- `浦島太郎｜AIが語る古の伝説【没入型アニメーション】`
- `桃太郎｜失われた鬼ヶ島の真実【AI完全映像化】`
- `かぐや姫｜月に帰った理由、誰も知らない【AI民話】`

### パターン B: 【パワーワード】+ 固有名詞 + 具体性

```
【パワーワード】[物語名] - [具体的な内容や問い]
```

例:
- `【衝撃】浦島太郎 - 玉手箱を開けた本当の理由`
- `【必見】桃太郎 - 鬼は本当に悪者だったのか`
- `【初公開】かぐや姫 - AIが再現した月の宮殿`

### パターン C: 数字 + 物語名 + テーマ

```
[数字]分で分かる[物語名]の[テーマ]
```

例:
- `5分で分かる浦島太郎の哲学【AI解説】`
- `浦島太郎3000年の孤独を10分で体験する`

### パターン D: 疑問形（好奇心を刺激）

```
[物語名]の[謎/問い]、あなたは知っていますか？
```

例:
- `浦島太郎はなぜ玉手箱を開けたのか？AIが解明`
- `竜宮城は実在した？日本最古の民話をAIで検証`

### パターン E: 英語圏向け（グローバル展開時）

```
[Character Name]: [Attribute] | [Culture] Mythology [AI qualifier]
```

例（海外チャンネルの実例パターンから）:
- `Urashima Taro: The Man Who Lost 300 Years | Japanese Folklore AI`
- `The Dragon Palace Story - Ancient Japan Reimagined with AI`
- `What It Was Like to Visit the Dragon Palace | AI Japanese Myth`

## 4. 海外 AI 創作動画チャンネルのタイトル実例

調査日: 2026-04-11

### 確認できた高再生チャンネルとタイトルパターン

| チャンネル名 | 再生数規模 | タイトルパターン |
|------------|-----------|----------------|
| **AI Historian** | 累計1億回近く | 歴史的事実 + 「What It Was Like to...」 |
| **Sleepless Historian** | 登録者60万人超 | 「Boring History for Sleep \| [詳細タイトル]」|
| **Legends Reimagined: AI Art and Mythology** | AI民話特化 | 「[神名]: The [属性] in Ancient [文化] Mythology」|

### 実際の高再生タイトル例（英語圏）

```
"Boring History for Sleep | How Medieval PEASANTS Survived..." → 230万再生
"What It Was Like to Visit a BR0THEL in Pompeii"
"Poseidon: The God of the Sea and Earthquakes in Ancient Mythology"
"The Aztec Creation Myth Mythology Stories AI Video"
"20 Short Stories About Cultures In Mythology AI Video"
```

### 英語圏タイトルから学べる法則

1. **「AI Video」「AI Art」を末尾に明示する** → 検索にヒットする + 視聴者の期待値を設定
2. **「What It Was Like to...」** → 体験型・疑似体験フレームは維持率が高い
3. **数字（10, 20）+ Short Stories** → シリーズ化しやすくアルゴリズムに好まれる
4. **「Reimagined」** → AI再解釈コンテンツの定番ワード
5. **キャラクター名 + コロン + 属性** → 固有名詞ファーストで検索に強い

## 5. 日本語タイトルのNGパターン

| NGパターン | 理由 | 改善案 |
|-----------|------|--------|
| 「浦島太郎の動画」 | キーワードが弱い。何が見られるか分からない | 「浦島太郎｜AIが語る竜宮城の真実」 |
| タイトルが40文字超 | スマホで省略される | 重要部分を前半28文字に集約 |
| 「AIで作りました」 | メリットが伝わらない | 「AIが完全映像化」で品質を訴求 |
| 「〜してみた」 | エンタメ系と混在して埋もれる | 民話+感情訴求に特化する |
| 釣りタイトル（内容と乖離） | 視聴維持率が落ちYouTubeに嫌われる | 内容と完全に一致させる |

## 6. タイトル生成チェックリスト（毎回使う）

動画タイトルを決める前に以下を確認:

- [ ] 先頭15文字以内に物語名（主要キーワード）が入っているか
- [ ] スマホで28文字以内に収まっているか（重要部分が省略されないか）
- [ ] パワーワードが1〜2個入っているか
- [ ] 感情フック（切ない・衝撃・真実・永遠 etc.）が入っているか
- [ ] 「AI」「AIが作った」「AI映像化」など制作方法が分かるか
- [ ] 内容と乖離した釣りタイトルになっていないか
- [ ] 候補を3つ作って一番クリックしたいものを選んでいるか
