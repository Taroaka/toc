# YouTube コミュニティ / CRM 戦略

最終更新: 2026-04-27
対象: `にわかのAI` のYouTubeコメント、概要欄、About、SNS導線

---

## 1. 方針

`にわかのAI` のCRMは、完成品を一方的に見せるのではなく、視聴者に `AIを一緒に育てる初期メンバー` になってもらう設計にする。

表で見せる人格:

- 投稿しているのは人間の制作者
- AIは完成品ではなく、育成中の制作システム
- 未完成な部分も隠さず、改善していく過程を見せる
- コメントは単なる感想ではなく、次の改善材料として扱う

裏で回収する価値:

- 応援してくれる視聴者
- 改善提案をくれる視聴者
- 次に見たい物語を出してくれる視聴者
- AI動画制作システムに関心がある見込み客

---

## 2. ブランドボイス

基本トーン:

- 軽い、近い、素直、未完成、でも前向き
- 金髪ホスト風の見た目は入口に使うが、言葉は誠実にする
- 上から教えるのではなく、見てくれた人に育ててもらう

使う言い回し:

- `見てくれてありがとう`
- `まだまだ改善中です`
- `ここから少しずつ育てていきます`
- `率直に教えてください`
- `次の作品づくりに反映します`
- `あなたの一言が改善材料になります`

避ける言い回し:

- `AIが全部作りました`
- `完全自動で完璧です`
- `まだ下手ですが許してください`
- `高評価とチャンネル登録お願いします` を固定コメントの主目的にする
- `批判でも何でもください`
- ホスト文脈が強すぎる `姫` `客` `指名して`

---

## 3. CRM導線

初期はリンク先を1つに集約する。

推奨リンク名:

`制作ノート・改善提案・相談はこちら`

フォームの分岐:

- `制作の裏側を受け取りたい`
- `改善提案・作ってほしい物語を送りたい`
- `AI動画制作システムについて相談したい`

接点ごとの役割:

| 接点 | 役割 |
|------|------|
| 固定コメント | 応援、改善提案、次回希望を集める |
| 概要欄 | 作品説明から制作システム理解へ進ませる |
| About | チャンネルの人間味と二重価値を説明する |
| プロフィールリンク | SNS共通のCRM入口にする |
| X投稿後リプ | 視聴後の制作ノート/改善提案/相談導線へ送る |

---

## 4. 固定コメントの標準型

固定コメントは外部リンクを置きすぎず、まずコメント欄での会話を作る。

推奨:

```text
ここまで見てくれてありがとうございます。正直、まだ未完成なところも多くて、公開するのは少し恥ずかしいです。

でも、AIに任せきりではなく、人間の自分が「どんな物語として届くか」を見ながら、1本ずつ育てていきたいと思っています。

この浦島太郎で、いちばん印象に残った場面、または「ここはもっと見たかった」という部分を1つだけ教えてください。次の作品づくりに反映します。
```

CRM寄り:

```text
にわかのAIの初期作品を見に来てくれてありがとうございます。

まだ粗い部分もありますが、ここから少しずつ、物語・映像・語りの精度を上げていきます。完成されたチャンネルというより、成長していく制作過程ごと見てもらえる場所にしたいです。

次に改善するなら、映像・語り・テンポのどれを一番直すべきだと思いましたか？短くても大丈夫なので、率直に教えてください。
```

---

## 5. 返信方針

- 最初の24時間は作品内容、改善点、次回希望に触れたコメントを最優先で返信する
- `ありがとうございます` だけで終わらせず、相手の言葉を1つ拾う
- 良い指摘には `次回の改善メモに入れます` と返す
- 次回候補には `その物語なら、どの場面を見たいですか？` と追加で聞く
- 厳しめの意見にも反論せず、どこでそう感じたかを聞く
- ハートは称賛よりも、具体的に見てくれたコメントに優先して付ける

返信例:

```text
具体的にありがとうございます。テンポは自分でも少し気になっていたので、次回は冒頭の入り方をもっと整理してみます。
```

```text
玉手箱の場面を拾ってもらえてうれしいです。あそこはもっと余韻を出せたかもしれません。次に昔話を作るなら、どの物語を見てみたいですか？
```
