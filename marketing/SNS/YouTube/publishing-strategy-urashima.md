# 浦島太郎 初回公開戦略

最終更新: 2026-04-12  
対象: `にわかのAI` の初回通常動画 `浦島太郎`

Context: new channel, no existing audience, only the video and thumbnail exist.

---

## 1. 推奨公開日と時刻

**推奨: 火曜日 19:00 JST**

理由:

- 2026年のYouTube配信タイミング系データでは、長尺動画は平日の午後〜夕方が強く、火・水・木が特に安定している。
- Sprout Socialの最新のYouTube timing guidanceでも、火曜夕方が強い。
- 日本向けの物語動画なら、19:00は仕事・学校が終わった後に見やすく、初動のクリックと視聴時間を取りやすい。

代替:

- 火曜が無理なら `木曜 19:00 JST`

---

## 2. Private / Unlisted / Public の流れ

**推奨フロー: `Private -> 最終確認 -> Scheduled Public`**

1. いったん `Private` でアップロードする。
2. タイトル、サムネイル、説明文、子ども向け設定、コメント設定、再生リストを確認する。
3. `Tuesday 19:00 JST` で `Scheduled Public` にする。
4. そのまま公開させる。

`Unlisted` は少人数の事前確認用だけに使う。公開初動の本番運用では使わない。リンク共有はできるが、公開面での発見性がないため、初回の集客導線としては弱い。

公開ページに表示される日付は `PST` 基準になるので、日付跨ぎの公開には注意する。

---

## 3. 公開30分前チェック

- 冒頭30〜60秒を再視聴する。
- タイトル先頭に `浦島太郎` が入っているか確認する。
- サムネイルがスマホでも読めるか確認する。
- `Not made for kids` になっているか確認する。
- コメントは `Hold potentially inappropriate comments for review` にしておく。
- スパムが来そうならリンク系のコメントを抑制する。
- 公開時刻が `火曜 19:00 JST` か確認する。
- 説明文の最初の2行が、動画内容と検索語をちゃんと含んでいるか確認する。
- 固定コメントの文面を用意する。
- 再生リストの紐づけを確認する。

---

## 4. 公開後30分チェック

- 公開ページを desktop と mobile で開いて確認する。
- 可能な warm surface があるなら、そこへだけ共有する。
- 事前に用意したコメントを pin する。
- 最初のコメントには早めに返信する。
- Studio で impressions, CTR, average view duration を見るが、すぐには結論を出さない。
- 明確な誤字・誤サムネ・壊れたリンクだけは即修正する。
- 何度も刷新しない。CTR の初期値はノイズが大きい。

---

## 5. Reach に効く設定

- `Visibility`: 公開時は `Public`
- `Audience`: `Not made for kids`
- `Comments`: 有効化して、保留モデレーションをかける
- `Block links`: 必要なら有効化
- `Title`: 検索語と興味喚起の両方を入れる
- `Thumbnail`: 高コントラスト、文字少なめ、実物と齟齬なし
- `Description`: 先頭2行に動画の核を置く
- `Category`: 実態に近いものを選ぶ
- `Upload defaults`: デスクトップ側で事前設定して、誤公開を防ぐ

もし利用可能なら、アカウントの検証や動画機能の有効化を先に済ませる。公開当日に機能制限で詰まるのを避けるため。

---

## 6. よくある失敗

- 準備完了の都合で、公開時刻を適当にする
- `Unlisted` のまま放置する
- `Made for kids` を誤って付ける
- コメントを切って、唯一の反応面を消す
- CTR が出る前にタイトルとサムネを何度もいじる
- `浦島太郎` をタイトルから見失う
- モバイルで確認しない
- 固定コメントを入れ忘れる
- 公開日が `PST` 基準でズレることを忘れる
