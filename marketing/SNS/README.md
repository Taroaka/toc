# SNS Marketing Guide

SNS 配下は、`にわかのAI` の配信・投稿・導線設計の正本を置く。

## 現在の主戦場

- YouTube: `YouTube/strategy.md` を北極星にする
- X / Instagram: 初期は YouTube 公開後の拡散導線として扱う
- リスト獲得: 概要欄・固定コメント・プロフィールリンクから、制作システムへの問い合わせへつなげる
- CRM: 視聴者を `AIを一緒に育てる初期メンバー` として扱い、コメント・改善提案・次回希望を集める

## 誰に何を届けるか

- AI好きのアーリーアダプターに、`物語名だけで動画が完成するAIエージェント制作システム` を見せる。
- 物語好きの一般視聴者に、`生成AIで再現した世界中のナラティブ` を届ける。
- 導入候補者に、`サービス手作業ではなくプログラムで制作されている` ことを伝える。

SNS の投稿文・概要欄・固定コメントは、単なる作品紹介で終わらせず、上の3点のどれかに必ず接続する。

初期の接点では、投稿者が人間であることを出す。`AIが全部作った完成品` ではなく、`人間の制作者がAIを育て、未完成な部分も改善していく過程` として見せる。
