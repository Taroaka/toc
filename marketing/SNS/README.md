# SNS Marketing Guide

SNS 配下は、`にわかのAI` の配信・投稿・導線設計の正本を置く。

## 現在の主戦場

- YouTube: `YouTube/strategy.md` を北極星にする
- X / Instagram: 初期は YouTube 公開後の拡散導線として扱う
- リスト獲得: 概要欄・固定コメント・プロフィールリンクから、制作システムへの問い合わせへつなげる
- CRM: 視聴者を `AIを一緒に育てる初期メンバー` として扱い、コメント・改善提案・次回希望を集める
- 横展開: YouTube の長尺動画を原液として、各SNS向けに切り出す。各プラットフォームで新しく発明し直さない。

## 誰に何を届けるか

- AI好きのアーリーアダプターに、`物語名だけで動画が完成するAIエージェント制作システム` を見せる。
- 物語好きの一般視聴者に、`生成AIで再現した世界中のナラティブ` を届ける。
- 導入候補者に、`サービス手作業ではなくプログラムで制作されている` ことを伝える。

SNS の投稿文・概要欄・固定コメントは、単なる作品紹介で終わらせず、上の3点のどれかに必ず接続する。

初期の接点では、投稿者が人間であることを出す。`AIが全部作った完成品` ではなく、`人間の制作者がAIを育て、未完成な部分も改善していく過程` として見せる。

YouTube は販売ページではない。YouTube では興味、信頼、リスト登録の理由を作り、セールスは LP / フォーム / DM / 個別相談に分離する。

## 投稿設計の共通原則

SNS では、最初に情報を説明しない。
まず相手の視点を決め、1秒で反応できる入口を作る。

| セグメント | 先に置く視点 | 避ける主語 | 次の導線 |
|------------|--------------|------------|----------|
| 物語好き | `この物語で何を失い、何が残るか` | AI技術の説明 | 本編視聴、次に見たい物語コメント |
| AI好き | `物語名だけでどこまで自動化できるか` | 完成品の自慢 | 制作ノート、改善ログ、裏側動画 |
| 制作者・企業 | `自分の企画も同じように映像化できるか` | 汎用的なツール紹介 | 相談フォーム、導入相談 |
| 初期応援者 | `自分の一言で作品が良くなるか` | 登録者数や実績 | 改善提案、コメント、次回希望 |

運用ルール:

- 冒頭は `結論` から入る。背景、制作方法、思想は後ろに回す。
- 投稿ごとに狙うセグメントを1つ決める。全員向けに丸めない。
- 論理説明より、驚き、切なさ、違和感、応援したさを先に置く。
- CTA は命令ではなく、相手が自分で選べる問いにする。
- 商品やシステムは主役ではなく、相手の判断を後押しする証拠として出す。
